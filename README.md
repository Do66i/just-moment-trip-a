# 서비스 소개

**여행 중 지출 내역을 보다 편리하게 관리할 수 있고 여행을 편하게 기록할 수 있도록 만든 웹사이트입니다.**

◇ 주요 기능 ◇
- 기능별 직관적 디자인 제공
- 여행별 가계부 / 다이어리 생성
- 여행 및 날짜별 가계부 및 다이어리 관리
- 사용 내역 통계 및 그래프 기능
- 소셜로그인 기능
- 서비스 이용자 간 채팅 기능
- 여행중 구입내역 Google Map pin기능
